This README is with regard to the 'sbstframe' package in this directory.  Various
subdirectories contain READMEs of their own for their respective packages.

ABOUT THIS DIRECTORY
--------------------
O pacote sbstframe contem toda a estrutura de funcionamento do framework SBSTFrame. O framework
é divido em três pacotes principais, são eles: "problem", "solution" e "results". O pacote "problem"
define o problema do usuário, por meio da comunição do framework com os arquivos de entrada (benchmarks). 
O pacote "solution" auxilia o usuário a configurar seu experimento selecionando ou definindo técnicas 
de busca e seus parâmetros. O pacote "results" permite ao usuário definir como serão capturados os 
resultados do exeprimento a ser executado.
